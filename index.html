HTML

HTML, singkatan dari HyperText Markup Language, adalah bahasa markup standar yang digunakan untuk membuat dan menyusun halaman web. HTML menyediakan struktur dasar untuk konten web dengan menggunakan berbagai elemen dan tag, yang memungkinkan browser untuk menampilkan teks, gambar, video, dan elemen lainnya dengan cara yang terstruktur.

CSS
